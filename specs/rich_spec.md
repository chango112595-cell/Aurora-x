# Rich Specification Example

## Function: add
- **Signature**: `add(a: int, b: int) -> int`
- **Examples**:
  - add(1, 2) = 3
  - add(0, 0) = 0
  - add(-1, 1) = 0
- **Post-conditions**:
  - Result equals sum of inputs

## Function: multiply
- **Signature**: `multiply(x: int, y: int) -> int`
- **Examples**:
  - multiply(2, 3) = 6
  - multiply(0, 5) = 0
- **Post-conditions**:
  - Result equals product of inputs
