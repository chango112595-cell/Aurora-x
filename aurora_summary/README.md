## üîß Essential Commands Quick Reference

### Get Started Immediately
```bash
# Clone and start Aurora (works on Windows/Linux/Mac)
git clone https://github.com/chango112595-cell/Aurora-x.git
cd Aurora-x
make install
make fire-up

# Talk to Aurora directly
./ask-aurora.sh "Hello Aurora, are you ready?"
./ask-aurora.sh "What languages do you know?"
./ask-aurora.sh "Generate Python code for fibonacci"
```

### System Status & Health
```bash
make status              # Check all services
make aurora-test         # Test Aurora Enhanced Core  
make language-test       # Test Language Grandmaster
curl http://localhost:5003/api/health  # Chat system health
```

### Emergency Procedures
```bash
make stop && make start  # Quick restart
make backup             # Create backup
make clean && make install  # Clean reinstall
git reset --hard pre-windows-compat-merge  # Emergency rollback
```
