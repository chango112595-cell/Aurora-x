üß™ 3. TESTING FIXES (Critical Past Issue)
Original Problem

Pack04 tests failed with:

ModuleNotFoundError: No module named 'tests.test_*'

Root Cause

All core/module.py files from all packs polluted sys.path.

Fix

Add conftest.py per pack that only adds that pack's path.

Remove global packs/conftest.py.

Remove all __init__.py inside pack test folders.

Create centralized runner:
run_pack_tests.py
